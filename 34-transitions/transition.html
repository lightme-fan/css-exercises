Transitions

Initial state

To start to make a use of transitions, you must set the element into hover.

Eg:
a:hover {...}

Target state

It really depends on what number you do set for the scale of trensition.

Eg:
a:hover {
transition: scale(0.5);
}

Transition property

There are several transition properties

  • Transition-duration
  • Transition-property
  • Transition-delay
  • Transition-timing-fuction

Difference between these properties

  • Transition-duration: sets the length of time a trnsition animation should take to complete.
  • Transition-delay: is the start of animation.
  • Transition-timing-fuction : is used to specify the speed of the transition.

Animatable css properties by using transtion

  • -moz-outline-radius
  • -moz-outline-radius-bottomleft
  • -moz-outline-radius-bottomright
  • -moz-outline-radius-topleft
  • -moz-outline-radius-topright
  • -webkit-line-clamp
  • -webkit-text-fill-color
  • -webkit-text-stroke

Multiple transition

I think it is possible to select more than property to transition at a time by using comma.

Eg:
a:hover {
transition-property: background, border-radius, color;
transition-duration: 0.4s, 0.8s, 0.5s;
}

Transition state

We have to make use of :hover.

a:hover {
transition-property: background, border-radius, color;
transition-duration: 0.4s, 0.8s, 0.5s;
}

The need of transition

At some point, transition is useful whe we want to decorate links or button.

The trouble of transition

Transition could create a problem because the servers expect for being fast.

The auto keyword being one of the end states

To stagger transition
