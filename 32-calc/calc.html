Calc function

Values of calc()

It can support these:

  • Number
  • Percentage
  • Frequency
  • Length
  • Time
  • Integer
  • Angle

The order of operations in this function

Addition or substraction must be inside of brakets(...) and need spaces aroud them.

Eg:
calc((2 + 3) * 3)

Type of units

It is possible to covert a unit to an another unit

Eg:
calc(50% - 1rem)

Spaces around plus and minus

We have to make the use of spaces around these operations, because the calculation you do really depends on them.

Eg:
Wrong
calc(50px -2rem)
Correct
calc(50px - 2rem)

Convertion of a value into negative

It is possible to change a value into a negative value

Eg:
calc(-100px + 3rem)
Or
calc(100px * -3rem)

Calc funciton and color

Calc can be used with colour, but it shoudn't be greater than 225.

Eg:
rgb(cal(var(--red) / 3), calc(var(--green) -50), var(--blue));
