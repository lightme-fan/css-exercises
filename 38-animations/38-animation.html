Keyframe animation

There is some special code to set up a keyframe animation. We have to use this code

  • @keyframe name {

    }

Special about animation

Animation is very different from other CSS. It is easy to understand.

Setting up keyframe animation

This is how we make the use of keframe animation:

@keyframe name {
0% {.....};
100% {....};
}
