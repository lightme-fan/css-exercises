CSS grid

Grid columns by default

By default, a grid has two columns which are:
  • grid-column-start
  • grid-column-end

Flexbox and Grid

Flexbox is only used to specify one dimension, either row or column, whereas Grid can be used for two dimensional layout. We can make the use of row and column at the same time.

Using css flexbox could be better than grid because flexbox is good to lay out elements in either single column or single row.
I would use grid than flexbox, as it is best to set out elements in either multiple rows or multiple columns.

A concise way to write out this:

grid-template-columns:10rem 10rem 10rem 10rem;
grid-template-columns: repeat(4, 10rem);
