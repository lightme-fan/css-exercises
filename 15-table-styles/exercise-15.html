Border-spacing

Table-layout: fixed

With this value, the layout of the table doesn't really care about the content, becuse the browser just uses the table's width.

Table-layout: auto

This value is default. The browser is automatically used to specify how the columns, rows and cells are put out.

Border-spacing can be used to make a space between line instead of using margin or padding.
