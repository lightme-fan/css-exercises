  • The property which is said above can be used
  • There are roughly seven Pseudo-elements in css which are:
    • ::after
    • ::before
    • ::first-letter
    • ::first-line
    • ::marker
    • ::placeholder
    • ::selection
  • When you want to style a special content which usually first letter or last letter, it is a good idea to use pseudo elements.

CSS counters

  • CSS Counters are used to adjust the appearance of the content in document by using numbers.
    Firstly, the value of the a property must be set as an initial value. On another word, it should be by default or 0. Once it is set initialized, the value can be increased or decreased.
  • To reset a counter in CSS, setting the value in to none or inherit or initial or unset is the best way to do that.
  • Counters as well are used to measure time and frequency; especially, designing sequential circuits by the flip flops.
