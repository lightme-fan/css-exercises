All the small properties which need to be known in CSS.

Cursor

CSS cursor property used to define cursor type when the mouse moves in an area or a link on the webpage.

These are the values of this property:
  • default
  • pointer
  • text
  • wait
  • help
  • progress
  • crosshair
  • move
  • url()

Pointer-events

Pointer-events property sets under what circumstances. A particular graphic element can become the target of pointer events.

Values
  • None
  • Auto
  • Only for SVG
    • all
    • stroke
    • fill
    • painted
    • visible

Outline

The outlines are generally used to highlight elements wich is as same as border

There are few properties to define outline
  • outline-color
  • outline-style
  • outline-width

Filter

Filter property is a way to apply the visual effect to the images.

Values

  • blur
  • brightness
  • contrast
  • drop-shadow
  • grayscale
  • hue-rotate
  • invert
  • opacity
  • saturate
  • sepia

Appearance

Object-fit

The object-fit CSS property sets how the content of a replaced element, such as an image or video, should be resized to fit its container.

Value

  • none
  • contain
  • cover
  • fill
  • scale-down

Scroll-snap-align

Scroll-snap-align gives us a way to snap scrolling to certain points as the user scrolls through a document.
