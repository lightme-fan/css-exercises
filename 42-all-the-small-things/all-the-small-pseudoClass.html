Pseudo-class selectors

:focus

This class is used to add special style to an element while the element has focus.

:focus-within

This pseudo-class represents an element that is itself matched by the :focus or has a descendant that is matched by :focus.

:visited

:visted represents links that the user has already visited

Example:
a:visited {
color: green;
}

:active

This is used to represent an element (such as a button) that is being activated by the user.

Example:
a:active {
color: red;
}

:target

This represents a unique element with an id matching the URL's fragment.

Example:
a:target {
border: 1px solid grey;
}

:disabled

This pseudo-class is used to represent any disabled element.

Example:
input:disabled {
background: #ccc;
}

:empty

:empty is used to show any element that has no children.

Example:
div:empty {
background: lime;
}

:not()

:not shows elements that do not match a list of selectors. Since it prevents specific items from being selected, it is known as the negation pseudo-class.

Example:
:not(p) {
color: blue;
}

:matches()

:matches() is a new CSS pseudo-class that allows to save time by grouping selectors together:

Example:
:matches(article, div.section, section) h2 {
font-size: 3em;
}

A pseudo-element

::first-letter

::first-letter is a pseudo-element which selects the first letter in the first line of a block-level element (such as a paragraph

).

Example:
p::first-letter {
font-family: "Gentium Book Basic", serif;
font-weight: bold;
color: grey;
}
