CSS transform

Function of transform

There are several functions we can use in transform

  • Scale()
  • Translate()
  • Rotate()
  • Matrix()
  • Skew()

Separated function of transform

You can include more than one transform on an element. To include more than one transform, simply separate them the tranform functions with spaces.

Moving axis of transform from its default position

In order to move the central pin of transform, we should use tranform-origin: top left;

Transform: translate

Position sometimes requires negative margin. Additionally, we have to use Top and Left whereas transform: translate() doesn't.

An element translated

The layout transform is similar to relative positioned.

Problem of overriding a transform

When we write more than transform properties in one specific selector, the first property you set doesn't work.

Transform 3D

When we want to use 3D, we have to use perspective

Eg:
body {
perspective: 800px;
}
div {
transform: rotate3d(1, 1, 1, 45deg);
}

Transform with box model

If we want to modify a box model by using transform, we should use transform-box.
