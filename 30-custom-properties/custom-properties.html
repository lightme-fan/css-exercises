Custom properties

  1. When we set a Custom Property's name, we must use double dashes --.
  2. In a Custom Property, is not used.
  3. The values wich can possibly be used depend on the main idea in a document.
    Eg:
    --number-of-people: 10;
    • Using :root or html of Custom properties is invaluable, when our decoration is applied in whole HTML document.
    • However :root is not always used if you want to limit the scope of your Custom properties.
  5. After we set a new Custom Property, in order to recall it, we can use var(...) function.
  6. Some names of changing a Custom Property:
    • --column
    • --color
    • --background
  7. Custom Properties are useful because it makes the css readable. They allow the values to be stored in one place, and then referenced in multiple places.
    Eg:
    :root { --color: red; }
    div { background-color: var(--color); }
  8. The only aspect of css syntax that we can use for Custom Properties:
    element {
    --name-of-property: value;
    }
