Overflow

Each of these properties below have their own default value.

  • Overflow:
    • visible
    • hidden
    • scroll
    • auto
  • Clip: auto
  • Clip-path: none
  • Text-overflow: clip

The need of these properties above.

When the content is over the edge of the box, these properties are useful to limit the line in order to fit them inside of the box.

These are the properties need to be uesd before the properties above.

  • Display
  • Max and min-width
  • Word-wrap

The browser's default value about overflow:

Ellipsis

Ellipsis is the display inside of the content. This decreases the amount of text displayed. If there is not enough space to display the ellipsis, it is clipped.

Text-overflow

Text-overflow property has difference from other properties because it sets the display of the content in order to be hidden, or clipped or ellipsis.
