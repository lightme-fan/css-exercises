Position

  1. The default value of position is static.
  2. Other properties which need to be used with non-default position value
    • Display
    • Line-height
    • Counter
  3. The display value that has been positioned is flex
  4. Position relative still maintains a box inside of the flow of a document.
  5. Position fixed
    The fixed value positions the relative to the initial containing block.
  6. Position sticky
    When the position is set as a sticky value, the reletive value is used to be offset the containg block and the nearest ancestor.
  7. The way to lock the coordinates of the element to its parent element is to set the parent element in position absolute.
  8. Absolute positioned layouts is pretty useful for containing block, which the dimensions and the position of the absolutely positioned are relative.
  9. The rules of stacking of the absolutely positioned that elements in the same physical space are:
    • Only descendant elements are considered when stacking is processed.
    • The whole element is considered in the stacking order of the parent stacking context.
  10. A css stacking context is an element that stifles a set of layers. This could be a root or the local of the stuking content.
