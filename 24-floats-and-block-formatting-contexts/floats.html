Floats and block formatting contexts

Floats

The default values of float and clear

  • float: none;
  • clear: none;

The possible values for clear

  • left
  • right
  • both

The rule associated with a floated item in terms of source order

When an element is floated, it is shifted to the left or right until it reach the edge of the content box, and then it will take place in a new line.

Layout with floats

It is possible to float other elements besides images for example paragaph.

In order to create a two-column layout with float, you can use both floated right and left with clear both.

BFC

  • BFC: Block formatting context
    This is a combination of floats, absolute positioned, and display blcok.
  • The properties which are used to create a BFC on the parent element are:
    • display: inline-block;
    • position: absolute;
    • overflow
